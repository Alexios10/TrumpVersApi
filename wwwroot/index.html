API Documentation

Guide to using the Staff Members, Trump Thoughts, and Trump Merch APIs

Base URL

The base url contains information about all available API's resources. All responses will return data in json. GET requests. 

    Base url: http://localhost:5068/api
        

     {
        "trumpThoughts": "http://localhost:5068/api/TrumpThoughts",
        "staffmembers": "http://localhost:5068/api/staffmembers",
        "TrumpMerch": "http://localhost:5068/api/TrumpMerch"
     }

Staff Members Endpoints

1. Get All Staff Members

Retrieve a list of all staff members.

Endpoint: GET /staffmembers

Example: http://localhost:5068/api/staffmembers

Response:

{
   "id": 1,
   "name": "Mike Pence",
   "image": "vice-president.png",
   "description": "",
   "title": "vice-president",
   "email": ""
}

2. Get Staff Member by ID

Retrieve a specific staff member by their ID.

Endpoint: GET /staffmembers/byid/{id}

Example: http://localhost:5068/api/staffmembers/byid/1

Response:

{
   "id": 1,
   "name": "Mike Pence",
   "image": "vice-president.png",
   "description": "",
   "title": "vice-president",
   "email": ""
}

3. Get Staff Member by Name

Retrieve a staff member by their name.

Endpoint: GET /staffmembers/byname/{name}

Example: http://localhost:5068/api/staffmembers/byname/Mike%20Pence

Response:

{
   "id": 1,
   "name": "Mike Pence",
   "image": "vice-president.png",
   "description": "",
   "title": "vice-president",
   "email": ""
}

4. Add a New Staff Member

Create a new staff member.

Endpoint: POST /staffmembers

Request Body:

{
   "name": "Sonny Perdue",
   "image": "",
   "description": "",
   "title": "Secretary of Agriculture",
   "email": ""
}

5. Update an Existing Staff Member

Update the details of an existing staff member.

Endpoint: PUT /staffmembers/{id}

Example: http://localhost:5068/api/staffmembers/1

Request Body:

{
   "name": "Sonny Perdue",
   "image": "",
   "description": "Updated description",
   "title": "Secretary of Agriculture",
   "email": "sonny.perdue@example.com"
}

6. Delete a Staff Member

Delete a staff member by their ID.

Endpoint: DELETE /staffmembers/{id}

Example: http://localhost:5068/api/staffmembers/1

Trump Thoughts Endpoints

1. Get All Trump Thoughts

Retrieve all Trump thoughts.

Endpoint: GET /trumpthoughts

Example: http://localhost:5068/api/trumpthoughts

Response:

{
   "id": 1,
   "name": "Trump",
   "thought": "Make America Great Again!",
   "dateCreated": "2024-12-01T12:34:56Z",
   "category": "Economy",
}

2. Add a Trump Thought

Create a new Trump thought.

Endpoint: POST /trumpthoughts

Request Body:

{
   "thought": "Winning bigly!",
   "dateCreated": "2024-12-08T14:00:00Z"
}

3. Update an Existing Trump Thought

Update the details of an existing Trump thought.

Endpoint: PUT /trumpthoughts/{id}

Example: http://localhost:5068/api/trumpthoughts/1

Request Body:

{
   "thought": "Greatest ever!",
   "category": "Politics",
   "dateCreated": "2024-12-08T15:00:00Z"
}

4. Delete a Trump Thought

Delete a Trump Thought by its ID.

Endpoint: DELETE /trumpthoughts/{id}

Example: http://localhost:5068/api/trumpthoughts/1

Trump Merch Endpoints

1. Get All Merchandise

Retrieve a list of all Trump merchandise items.

Endpoint: GET /trumpmerch

Example: http://localhost:5068/api/trumpmerch

Response:

[ 
    {
       "id": 1,
       "name": "MAGA Hat",
       "price": 19.99,
       "image": "maga-hat.png"
    },
    {
       "id": 2,
       "name": "Trump Mug",
       "price": 14.99,
       "image": "trump-mug.png"
    }
]

2. Add a Merchandise Item

Create a new merchandise item.

Endpoint: POST /trumpmerch

Request Body:

{
   "name": "Trump T-Shirt",
   "price": 24.99,
   "image": "trump-tshirt.png"
}

3. Update an Existing Merchandise Item

Update the details of an existing merchandise item.

Endpoint: PUT /trumpmerch/{id}

Example: http://localhost:5068/api/trumpmerch/1

Request Body:

{
   "name": "Trump T-Shirt",
   "price": 29.99,
   "image": "new-trump-tshirt.png"
}

4. Delete a Merchandise Item

Delete a Trump merchandise item by its ID.

Endpoint: DELETE /trumpmerch/{id}

Example: http://localhost:5068/api/trumpmerch/1
